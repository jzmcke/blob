# JavaScript Blob Decoder Tests

This directory contains JavaScript tests for the blob decoder.

## Setup

Install dependencies:
```bash
cd test/js
npm install
```

This will install Jest for running the tests.

## Running Tests

From the `test/js` directory:
```bash
npx jest blob.test.js
```

Or run all blob tests (including JS tests) from the blob repository root:
```bash
python run_all_tests.py
```

## Dependencies

- Node.js 16+ 
- Jest 27 (installed via npm)

The test validates that the JavaScript blob decoder (`js/blob.js`) correctly decodes blob files generated by the C implementation.
